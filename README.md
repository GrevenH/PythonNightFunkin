# Python Night Funkin

### Version: `v0.3-beta`

It is an _amateur_ project.

Based on [_**Friday Night Funkin**_](https://fridaynightfunkin.ru) from [_**Ninjamuffin99**_](https://ninja-muffin24.itch.io)

The release will include:
- **Modificable** levels
- **Modificable** players
- **PC/Mobile** support
- **Launcher** for _PC_
- _Local_ **multiplayer**
